# launch.ne

A scalable, upgrade-friendly Next.js boilerplate with type-safe envs, shadcn/ui, Biome, performance optimizations, and a scalable dara folder pattern and more.

This project exists to reduce the time spent configuring tools and let you start building real features immediately.
